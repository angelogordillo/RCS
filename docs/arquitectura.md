# Arquitectura Inicial RCS México

## Capas
1. Presentación: interfaz web (dashboard operativo).
2. Servicios: API de red logística (nodos, rutas, alertas).
3. Datos: catálogo logístico, eventos operativos y métricas.

## Entidades clave
- Nodo: proveedor, planta, puerto, CEDIS, cliente.
- Ruta: origen, destino, modo, costo, tiempo y riesgo.
- Evento: incidente que afecta flujo (clima, seguridad, aduana, capacidad).

## KPIs sugeridos
- OTIF (On Time In Full)
- Tiempo promedio de tránsito
- Costo por tonelada-km
- Índice de riesgo por corredor
- Utilización de capacidad por nodo
